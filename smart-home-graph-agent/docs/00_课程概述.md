# 智能家居图谱代理 - 课程概述

## 📚 课程介绍

本课程通过一个完整的智能家居助手项目，系统性地讲解现代AI Agent开发的核心技术栈：

- **LangChain** - LLM应用开发框架
- **LangGraph** - 多步骤Agent工作流引擎
- **Neo4j** - 图数据库
- **知识图谱** - 结构化知识表示
- **GraphRAG** - 基于图的检索增强生成

---

## 🎯 学习目标

完成本课程后，学生将能够：

### 知识层面
1. 理解知识图谱与传统数据库的区别
2. 掌握GraphRAG与传统RAG的差异
3. 理解AI Agent的设计模式
4. 掌握LangGraph状态机工作流设计

### 技能层面
1. 使用Cypher查询语言操作Neo4j
2. 设计和实现GraphRAG检索策略
3. 使用LangChain构建LLM应用
4. 使用LangGraph实现多步骤Agent

### 应用层面
1. 独立设计知识图谱Schema
2. 实现端到端的RAG系统
3. 构建可解释的AI Agent
4. 调试和优化Agent性能
